The Process
From Raw Data
to Political Intelligence
01
Corpus Fetch
2,000–4,000 articles per symbol from Media Cloud (English-language Anglo-American sources). GitHub Actions, no local setup required.
02
D Computation
PE via LDA (K=10, JSD) + ICI via sentence embeddings (MiniLM-L12). Bootstrap n=200 for publication-grade confidence intervals.
03
V, A, N Auto
Viral Velocity from λ calibration. Affective Weight via VADER sentiment. Network Coverage from article date distribution.
04
SRM Score
SRM = V × A × e−λD × N. Full result JSON + ZIP artifact uploaded automatically. LOOCV validates out-of-sample generalization.
